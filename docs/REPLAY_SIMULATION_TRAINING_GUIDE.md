# APRS Command Replay, Simulation, and Training Guide

Replay, simulation, and training features let you test workflows without live transmit.

## How to Open

Open Replay, Simulation, or Training in the lower-right feature tab area.

## What It Does

- Replay reviews recorded packet data.
- Simulation creates safe sample stations and packets.
- Training provides controlled practice scenarios.

## Step-by-Step Use

1. Open Replay, Simulation, or Training.
2. Choose a sample or local file when available.
3. Start the session.
4. Watch the map, station list, packet monitor, logs, alerts, and weather panels.
5. Stop the session before switching data sets.

## Safe Defaults

Replay, simulation, and training cannot transmit. Data is source-tagged so it remains separate from live APRS-IS or RF data.

## Common Problems

- Replay not loading: check file path and file format.
- Simulation not visible: check filters and station list.
- Training state stale: reset the scenario.

## Troubleshooting

Use the packet monitor and logs to confirm replay or simulation packets are being processed.

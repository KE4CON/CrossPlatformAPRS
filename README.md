## FieldComms EmComm Field Server

**https://github.com/KE4CON/FieldComms-IMS**

An off-grid emergency communications server for Raspberry Pi. A self-contained web application providing ICS forms, net control logging, APRS tactical map, resource board, roster, Winlink, NTS radiogram generator, and more вЂ” all running locally with no internet required.

- 26-page web frontend (HTML/CSS/JavaScript)
- Python backend services (FCC lookup, health monitor, ICS platform, dead man's switch)
- systemd service units and udev rules for automatic startup
- Install script for clean Raspberry Pi deployment
- Designed to integrate with GrayWolf for APRS
